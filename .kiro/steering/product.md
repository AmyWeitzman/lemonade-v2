# Product Overview

lemonade-v2 is a game project currently in the design phase. The project includes game design documentation covering game elements and instructions.

## Current Status
- Early development stage
- Design documents available (Game_Elements.xlsx, Game_Instructions.docx)
- Implementation not yet started

## Key Artifacts
- Game_Elements.xlsx: Contains game mechanics, elements, and data structures
- Game_Instructions.docx: Contains gameplay rules and instructions
