# Project Structure

## Root Directory
- `.git/` - Version control
- `.kiro/` - Kiro AI assistant configuration and steering rules
- `.vscode/` - VSCode workspace settings
- `Game_Elements.xlsx` - Game design elements (not tracked in git)
- `Game_Instructions.docx` - Game instructions (not tracked in git)
- `README.md` - Project documentation

## Conventions
- Design documents (Excel/Word files) are excluded from version control
- Environment configuration stored in `.env` (not tracked)
- Node modules excluded from version control

Implementation structure to be defined as development progresses.
